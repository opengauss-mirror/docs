# gms_utility概述

gms_utility是一个基于openGauss的插件，为用户提供了各种实用的程序和函数。目前支持以下23个接口：

- `ANALYZE_DATABASE`
- `ANALYZE_SCHEMA`
- `CONONICALIZE`
- `COMMA_TO_TABLE`
- `COMPILE_SCHEMA`
- `DB_VERSION`
- `EXEC_DDL_STATEMENT`
- `EXPAND_SQL_TEXT`
- `FORMAT_CALL_STACK`
- `FORMAT_ERROR_BACKTRACE`
- `FORMAT_ERROR_STACK`
- `GET_CPU_TIME`
- `GET_ENDIANNESS`
- `GET_HASH_VALUE`
- `GET_SQL_HASH`
- `GET_TIME`
- `NAME_RESOLVE`
- `NAME_TOKENIZE`
- `OLD_CURRENT_SCHEMA`
- `OLD_CURRENT_USER`
- `IS_BIT_SET`
- `IS_CLUSTER_DATABASE`
- `TABLE_TO_COMMA`
