# GAUSS-02661 -- GAUSS-02670<a name="ZH-CN_TOPIC_0302073484"></a>

GAUSS-02661: "SET %s takes only one argument"

SQLSTATE: 22023

错误原因：当前GUC变量不支持指定多个参数值。

解决办法：请检查SET命令确保每次只指定一个参数值。
