# GS\_MASKING\_POLICY\_ACTIONS<a name="ZH-CN_TOPIC_0306525313"></a>

GS\_MASKING\_POLICY\_ACTIONS系统表记录动态数据脱敏策略中相应的脱敏策略包含的脱敏行为，一个脱敏策略对应着该表的一行或多行记录。需要有系统管理员或安全策略管理员权限才可以访问此系统表。
