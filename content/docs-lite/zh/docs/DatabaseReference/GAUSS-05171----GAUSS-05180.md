# GAUSS-05171 -- GAUSS-05180<a name="ZH-CN_TOPIC_0302073497"></a>

GAUSS-05172: "failed to get pooled connections"

SQLSTATE: 53000

错误原因：从pooler manager线程获取连接失败。

解决办法：查看报错日志附近pooler manager的日志确定错误根因，并根据相应日志尝试修复。
