# GS\_SESSION\_MEMORY<a name="ZH-CN_TOPIC_0289900076"></a>

GS\_SESSION\_MEMORY视图统计Session级别的内存使用情况，包含执行作业在数据节点上gaussdb线程和Stream线程分配的所有内存。当GUC参数enable\_memory\_limit的值为off时，本视图不可用。
