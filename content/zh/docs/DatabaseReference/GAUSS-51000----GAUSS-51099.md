# GAUSS-51000 -- GAUSS-51099<a name="ZH-CN_TOPIC_0302073494"></a>

GAUSS-51000: "THP services must be shut down."

SQLSTATE: 无

错误原因：THP服务没有关闭。

解决办法：关闭THP服务。

GAUSS-51001: "Failed to obtain THP service."

SQLSTATE: 无

错误原因：获取THP服务信息失败。

解决办法：检查THP文件是否存在。

GAUSS-51002: "Failed to close THP service."

SQLSTATE: 无

错误原因：关闭THP服务失败。

解决办法：检查THP文件是否存在。
