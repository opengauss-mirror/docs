# GAUSS-50900 -- GAUSS-50999<a name="ZH-CN_TOPIC_0302073456"></a>

GAUSS-50900: "The NTPD service is not installed."

SQLSTATE: 无

错误原因：时钟服务未安装。

解决办法：安装时钟服务。

GAUSS-50901: "The NTPD service is not started."

SQLSTATE: 无

错误原因：时钟服务未启动。

解决办法：启动时钟服务。

GAUSS-50902: "The system time is different."

SQLSTATE: 无

错误原因：系统时间不一致。

解决办法：更改系统时间使其保持一致。
