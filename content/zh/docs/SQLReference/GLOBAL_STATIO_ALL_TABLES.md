# GLOBAL\_STATIO\_ALL\_TABLES

GLOBAL\_STATIO\_ALL\_TABLES视图将包含各节点的数据库中每个表（包括TOAST表）的I/O的统计。

示例：
```sql
openGauss=# select * from DBE_PERF.GLOBAL_STATIO_ALL_TABLES;

-----------+-------+--------------------+-------------------------------+----------------+---------------+---------------+--------------+-----------------+----------------+----------------+---------------
```
