# GAUSS-03051 -- GAUSS-03060<a name="ZH-CN_TOPIC_0302072943"></a>

GAUSS-03051: "invalid input syntax for type point: '%s'"

SQLSTATE: 22P02

错误原因：point类型的数据非法。

解决办法：检查输入参数。
