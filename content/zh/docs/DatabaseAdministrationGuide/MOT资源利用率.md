# MOT资源利用率<a name="ZH-CN_TOPIC_0280525099"></a>

在4路96核512GB RAM的x86服务器上测试的资源利用率如下所示。MOT能够高效持续消耗几乎所有可用的CPU资源。例如，192核390万tpmC的CPU利用率几乎达到100%。

-   tmpC：每分钟完成的TPC-C事务数以橙色条柱和左侧的Y轴**tpmC**表示。
-   CPU利用率（%）：CPU利用率由蓝色线条和右侧的Y轴**CPU%**表示。
