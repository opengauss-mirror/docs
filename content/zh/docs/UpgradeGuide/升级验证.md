## 升级版本查询

本节介绍版本查询的具体操作。

### 验证步骤<a name="section117172026191017"></a>

1. 以数据库用户（如omm）登录节点，source环境变量。

2. 执行如下命令查看所有节点的版本信息。

   ```
   gs_ssh -c "gsql -V"
   ```

## 检查升级数据库状态

本节介绍数据库状态查询的具体操作。

### 验证步骤<a name="section117172026191017"></a>

1. 以数据库用户（如omm）登录节点。

2. 执行如下命令查看数据库状态。

   ```
   gs_om -t status
   ```

   查询结果的cluster\_state为Normal代表数据库正常。
